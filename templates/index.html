How to Convert Salinity → Ionic Strength

Formula: I = 0.5 × Σ(cᵢ zᵢ²)

Examples:

  • NaCl → I = c
  • KCl → I = c
  • CaCl₂ → I = 3c
  • MgCl₂ → I = 3c
  • Na₂SO₄ → I = 3c

For mixed brines, calculate ionic strength from all ion concentrations separately.
